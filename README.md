## Haptic Collision Simulator

A C++ project that simulates object collisions and generates corresponding haptic feedback in real time. The goal of this project is to explore how physical interactions in a virtual environment can be translated into force responses suitable for haptic devices.

### Run

Clone the repository and open the file in Visual Studio (or any other C++ IDE)

```
git clone https://github.com/smaaroof/Haptic-Collision-Simulator.git
```
